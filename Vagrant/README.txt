Prerequesite:
Install Virtualbox  (https://www.virtualbox.org/wiki/Downloads)
Install Vagrant	(https://www.vagrantup.com/downloads.html)



Vagrant is used to provision a virtual machine downloaded from the internet
and install programs in this case a postgressql to avoid the hassle to do it manually.
THe whole configuration is done in the Vagrantfile.The VM has the advantage to create
snapshots in order to set ack the database to a specific state.


To open the command line in directory: SHIFT + Right click
To start the VM execute in command line: vagrant up
to shut the VM down execute: vagrant halt
